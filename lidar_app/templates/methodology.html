Carbon estimation workflow — how results are computed

Processing Pipeline

Six-step workflow from raw LiDAR point cloud to per-tree carbon stock estimates

STEP 01
DTM & CHM Generation
Ground (Class 2) points used for DTM via minimum-elevation binning.
Vegetation (Classes 3–5) used for DSM via maximum-elevation binning.

CHM = DSM − DTM CHM clipped to 0 to remove negative values.
Class 2 → Ground elevation
STEP 02
Tree Top Detection
Local maxima in the CHM grid are found using scikit-image.
Each detected peak = one candidate tree top.

peak_local_max(CHM, min_distance=3, threshold_abs=2)
scikit-image peak_local_max
STEP 03
Crown Delineation
Marker-controlled watershed segmentation on the inverted CHM.
Each region = individual tree crown.

watershed(−CHM, markers, mask=CHM > 2) Crown Ø = 2 × √(Area / π)
scipy watershed segmentation
STEP 04
DBH Estimation
Diameter at breast height estimated from tree height and crown diameter using an empirical allometric model.

DBH = 1.3 + 0.6 × H + 0.3 × Crown Ø No field measurement required.
Height + Crown → DBH
STEP 05
AGB — Chave et al.
Above-ground biomass via the pantropical allometric equation from Chave et al. (2014).

AGB = 0.0673 × (ρ × D² × H)⁰·⁹⁷⁶ Wood density ρ = 0.6 g/cm³ (mixed tropical default).
Chave et al. 2014 pantropical
STEP 06
Carbon & CO₂
IPCC standard conversion factors applied to AGB.

Carbon = AGB × 0.47 CO₂ equivalent = Carbon × 3.67 Factor 0.47 = carbon fraction of dry biomass.
Factor 3.67 = CO₂/C molecular mass ratio.
IPCC standard factors

Key Equations

Mathematical foundations used in the carbon pipeline

Canopy Height Model
CHM = DSM − DTM
DSM = maximum Z in each grid cell from vegetation points. DTM = minimum Z from ground points. Represents vegetation height above bare earth.
DBH from LiDAR metrics
DBH (cm) = 1.3 + 0.6·H + 0.3·CD
H — tree height (m)
CD — crown diameter (m)
Above-Ground Biomass — Chave et al. 2014
AGB = 0.0673 × (ρ · DBH² · H)⁰·⁹⁷⁶
ρ — wood density = 0.6 g/cm³
DBH — diameter at breast height (cm)
H — total height (m)
AGB — above-ground biomass (kg)
Carbon & CO₂ Conversion
C = AGB × 0.47    CO₂ = C × 3.67
0.47 — IPCC carbon fraction
3.67 — CO₂/C ratio (44/12)
C — carbon stock (kg)
CO₂ — CO₂ equivalent (kg)
📐 Alternative Methods
Method 1 ✓ Applied Tree-Level Allometric Per-tree Height → DBH → AGB → Carbon. Best for forests with clear individual crowns. Produces per-tree inventory with spatial coordinates.

Method 2 Area-Based Grid Mapping Grid cells (10×10 m). Regression: AGB = a + b·H + c·CC. Produces AGB density maps without individual tree detection.
